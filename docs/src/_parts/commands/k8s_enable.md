## k8s enable

Enable core cluster functionalities

### Synopsis

Enable one of network, dns, gateway, ingress, local-storage, load-balancer, metrics-server.

```
k8s enable <functionality> [flags]
```

### Options

```
  -h, --help   help for enable
```
