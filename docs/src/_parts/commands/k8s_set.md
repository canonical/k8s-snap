## k8s set

Set cluster configuration

### Synopsis

Configure one of network, dns, gateway, ingress, local-storage, load-balancer, metrics-server.
Use `k8s get` to explore configuration options.

```
k8s set <functionality.key=value> ... [flags]
```

### Options

```
  -h, --help   help for set
```
